Place your hero image here with the filename `bengkel.jpg`.

Recommended:
- Size: ~1920×900 or similar wide aspect ratio
- Optimize/compress for web (JPEG, quality 70-85)

After adding the image, open the page:

```powershell
start 'd:\Sekolah\Ulangan\bengkel.html'
```

If you want a different filename, update the `background-image` URL in `bengkel.html` (inside the `#home .hero` section).

Brand/logo image:
- Place your header brand image in this folder with the filename `brand.png`.
- Recommended size: height ~40px (width auto). Use PNG or SVG for logos.
- If you use a different name, update the `src` in the header `.brand-logo` image tag in `bengkel.html`.

Service images (place these to make Layanan Kami show photos):
- `service-rutin.jpg` — untuk kartu "Service Rutin" (rekomendasi: 800×800, crop square)
- `perbaikan-mesin.jpg` — untuk kartu "Perbaikan Mesin"
- `ganti-ban.jpg` — untuk kartu "Ganti Ban & Rem"

Tips:
- Simpan ukuran gambar yang konsisten (square) dan optimalkan JPEG/WEBP untuk web.
- Jika Anda ingin memakai format lain, sesuaikan nama file pada `bengkel.html`.
